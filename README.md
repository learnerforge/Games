# Games

## Description

just an simple games stored online ready to play

---

## Tech Stack

HTML, CSS, JavaScript, LocalStorage, GitHub Pages

## Features

- Multiple browser games in one website
- Game cards with Play buttons
- Single-player and same-device 2-player modes
- Tic Tac Toe, Rock Paper Scissors, Memory Match, Snake, and Connect Four
- Player name input
- Local score tracking using browser storage
- Leaderboard for each game
- Restart and back-to-home buttons
- Responsive design for mobile and desktop
- Dark, blue, and dust theme support

## Installation

```bash
Clone the GitHub repository
Open the project folder in VS Code
Open index.html in a browser
Or use the Live Server extension in VS Code
Deploy the project using GitHub Pages
```

## Usage

```bash
Open the website in any browser
Enter your player name
Choose a game from the home page
Click the Play button
Play the selected game
Use Restart to start again
Use Back to Home to choose another game
Check your score and leaderboard after playing
```

## Target Audience

Students, casual gamers, beginners learning web games, friends playing same-device multiplayer, and users who want quick browser-based mini games.
